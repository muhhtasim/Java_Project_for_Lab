# Java Calculator Project for OOP Lab

This is a **Java-based Calculator Application** developed as a lab project for the Object-Oriented Programming (OOP) course.  
The project demonstrates the practical application of OOP concepts like **Encapsulation, Abstraction, Inheritance, and Polymorphism** in a real-world scenario.

---

## **Project Overview**
This calculator is designed to provide a simple and user-friendly interface similar to a standard handheld calculator.  
It performs basic arithmetic operations like **addition, subtraction, multiplication, division**, as well as **percentage, square root, positive/negative toggle**, and supports **decimal calculations**.

The project is implemented using **Java Swing GUI**, making it interactive and visually appealing.

## **OOP Concepts Applied**
- **Encapsulation:** Private variables and public methods to access them  
- **Abstraction:** Hides complex logic from the user, exposing only required operations  
- **Inheritance & Polymorphism:** Logical structure allows easy reuse and extension  
- **Modularity:** Separate methods for calculations, display updates, and event handling
